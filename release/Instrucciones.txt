Contenido:
  1) GoliathTA.bak : Backup de la base de datos de Goliath T&A
  2) GoliathTAServer.exe : Servidor a ser instalado en el servidor de aplicaciones
  3) GoliathTAClient.exe : Cliente del programa a ser instalado por el administrador, Jefe de Recursos Humanos o Jefes de Departamentos
  4) *.INI : Archivos de configuración de los programas anteriores. Se deben actualizar los datos de conexión con el servidor de bases de datos (GoliathTAServer.ini) o el DNS donde se instala el servidor GoliathTA (GoliathClient.ini).
  5) GoliathTA - Manual de Usuario : Borrador del Manual de Usuario. No está actualizado hasta las últimas modificaciones de la versión actual del programa.
  6) spImportEvents.sql : Código del procedimiento almacenado que se ejecuta periódicamente en una tarea de SQL Server para importar los eventos de acceso desde la base BioAdmin. Debe actualizarse el nombre de la base de datos (de ser necesario).
  7) Job Importar Eventos.sql : Código a ejecutar en el Analizador de Consultas para crear la tarea periódica que sincroniza ambas bases de datos (BioAdmin -> GoliathTA). Debe actualizarse el nombre del servidor SQL Server y credenciales.

Verificación:
  Una vez instalado todo, cada minuto se ejecutará el procedimiento spImportarEventos de GoliathTA para añadir los eventos de acceso concedido registrados en la base de datos BioAdmin.
